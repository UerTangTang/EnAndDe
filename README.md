# EnAndDe加解密代理工具

- 📚 可能网上有比较好的工具来进行加解密配合抓包工具，但是由于我学习经验不够最后没有找到，正好来制作这款工具锻炼一下自已
- 👋 用于一键启动上下游代理，方便对数据包和响应包进行加解密处理，然后可以方便配合Burp进行测试
- ❤️ anli文件夹里面有加解密方法书写格式，可能代码不太严谨，还请大家见谅！
